# i18n Audit

Multi-language translation audit toolkit.

---

## 🚀 Packages

| Language | Package | Status |
|----------|---------|--------|
| Rust     | i18n-audit-rust | Stable |
| JS/TS    | @i18n-audit/javascript | Beta |
| PHP      | i18n-audit/php | Stable |

---

## 📦 Installation

### Rust
cargo install i18n-audit

### JavaScript
npm install @i18n-audit/javascript

### PHP
composer require i18n-audit/php
